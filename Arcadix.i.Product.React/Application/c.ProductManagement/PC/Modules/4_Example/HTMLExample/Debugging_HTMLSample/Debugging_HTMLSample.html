Documentation for Debugging

Debugging is the process of finding and fixing errors within a script. All modern browsers and most other environments support debugging tools – a special UI in developer tools that makes debugging much easier. It also allows to trace the code step by step to see what exactly is going on.

Open developer tool in chrome browser

To open DevTools, you can right-click anywhere on the page and select inspect element or you can choose the “tools > developer tools” option from the top right menu. or you can use ctrl+shift+I shortcut key.

Change DevTools Positions

Use ctrl+shift+D to change devtool position or you can use Dock options:

  1. unlock into saperate window
  2. dock to left
  3. dock to right
  4. dock to bottom
Simulate Mobile Devices with Device Mode

Use Device Mode to approximate how your page looks and performs on a mobile device.

Click Toggle Device Toolbar to open the UI that enables you to simolate a mobile viewport.

Open the Elements panel to inspect the DOM or CSS

When you want to inspect a DOM node's styles or attributes, right-click the element and select Inspect.

Or press Command+Option+C (Mac) or Control+Shift+C (Windows, Linux, Chrome OS).

Quick-edit a HTML Element
  1. Select the “Elements” panel.
  2. Choose a DOM element within the Elements Panel editor.
  3. Right click on the selected tag and select edit as HTML option.

Once you are finish editing click outside of editor.

View an element's CSS
  • Right-click on the element and select Inspect. The Elements panel of DevTools opens. The Inspect Me! element is highlighted in the DOM Tree.
  • The Styles tab on the Elements panel lists the CSS rules being applied to whatever element is currently selected in the DOM Tree
Add a CSS class to an element

Use the Styles tab to see how an element looks when a CSS class is applied to or removed from an element.

  • Right-click on element and select Inspect.
  • Click .cls DevTools reveals a text box where you can add classes to the selected element.
  • Type class_name in the Add new class text box and then press Enter. A checkbox appears below the element, where you can toggle the class on and off. If the element had any other classes applied to it, you'd also be able to toggle them from here.
Console Overview
Viewing logged messages

Use the Styles tab to see how an element looks when a CSS class is applied to or removed from an element.

To log a message, you insert an expression like console.log('Hello, Console!') into your JavaScript. When the browser executes your JavaScript and sees an expression like that, it knows that it's supposed to log the message to the Console.

Running JavaScript

You can run JavaScript in the Console to interact with the page that you're inspecting.

  • Open devtool > console tab
  • Here you can see console message
  • You can write and execute any javascript code in console
