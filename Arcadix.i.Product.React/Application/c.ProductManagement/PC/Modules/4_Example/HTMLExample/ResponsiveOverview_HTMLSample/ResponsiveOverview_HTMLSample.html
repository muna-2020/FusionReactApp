Responsive Overview
Setting The Viewport

Add this meta tag inside head tag of all HTML pages. This will set the viewport of your page, which will give the browser instructions on how to control the page's dimensions and scaling.

                <meta name="viewport" content="width=device-width, initial-scale=1.0" >
Responsive Images

Using the width or max-width Property

                <img src="img.jpg" style="max-width:100%;height:auto;">
Responsive Text Size

The text size can be set with a "vw" unit, which means the "viewport width".

That way the text size will follow the size of the browser window:

                font-size: 2vw;
Media Queries

use CSS media query to make webpage responsvie for all devices

                @media screen and (min-width: 360px) and (max-width: 728px) {
                    .article{
                        width:100%;
                    }
                }
                @media screen and (min-width: 728px){
                    .article{
                        width:70%;
                    }
                }
