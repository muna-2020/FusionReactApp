CSS Vendor Prefixes

CSS vendor prefixes, also sometimes known as or CSS browser prefixes, are a way for browser makers to add support for new CSS features before those features are fully supported in all browsers.

Common Prefixes
Browser Prefixes
Chrome, Safari,
newer versions of Opera,
almost all iOS browsers including Firefox for iOS;
basically, any WebKit based browser		 -webkit-
Firefox -moz-
old pre-WebKit versions of Opera -o-
Internet Explorer and Microsoft Edge -ms-
Sample Example 
                    -webkit-transition: all 4s ease;
                    -moz-transition: all 4s ease;
                    -ms-transition: all 4s ease;
                    -o-transition: all 4s ease;
                    transition: all 4s ease;
