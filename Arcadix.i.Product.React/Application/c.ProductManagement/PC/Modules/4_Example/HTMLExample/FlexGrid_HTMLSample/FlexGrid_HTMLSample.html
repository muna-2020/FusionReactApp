Flexbox and Grid
Flexbox
Parent Element (Container)
HTML 
                                <div class="box-container">
                                    <div>1</div>
                                    <div>2</div>
                                    <div>3</div>
                                </div>
CSS 
                                /* The box container becomes flexible by setting the display property to flex */
    
                                .box-container{
                                    display: flex;
                                }
Output
1
2

Perfect Centering
HTML 
                                <div class="box-container">
                                    <div>1</div>
                                </div>
CSS 
                                /* Set both the justify-content and align-items properties to center, and the flex item will be perfectly centered */
    
                                .box-container{
                                    display: flex;
                                    justify-content: center;
                                    align-items: center;
                                }
Output
1

Grid
Display Property

                                <div class="grid-box-container">
                                    <div class="grid-item">1</div>
                                    <div class="grid-item">2</div>
                                    <div class="grid-item">3</div>  
                                    <div class="grid-item">4</div>   
                                    <div class="grid-item">5</div>  
                                    <div class="grid-item">6</div>

                                /* An HTML element becomes a grid container when its display property is set to grid or inline-grid */
    
                                .grid-box-container {
